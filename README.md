# じゃんけんゲーム

## ゲームの内容
企画内容:シンプルかつ誰でも理解できる普遍性と運に委ねる面白さを持つ2DじゃんけんゲームをUnityで製作しました　理由:じゃんけんというコンテンツは年齢や経験を問わず世界中で親しまれており、複雑なルールや操作を必要としない普遍的な面白さを持っていて、また、勝敗の結果が運に委ねられる要素が強いため、プレイヤーが繰り返し遊びたくなるような中毒性も期待できると考えたのでじゃんけんゲームを製作しました。
